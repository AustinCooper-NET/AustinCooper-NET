## 🚀 Featured Networking Projects

### 📡 Cisco Packet Tracer Labs
A structured progression of networking labs focused on building practical skills in enterprise-style network design and troubleshooting.

**Core Phases:**
- **Basic LAN** — Learned Layer 2 switching behavior, MAC address learning, and static IP configuration.
- **Secure SOHO Network** — Configured DHCP services, wireless access, and basic network segmentation.
- **Enterprise Gateway** — Implemented static routing and default gateway behavior between LAN and WAN.
- **Enterprise LAN/WAN** — Built WAN connectivity between internal networks and ISP-style routing environments.
- **VLAN & Inter-VLAN Routing (ROAS)** — Designed VLAN segmentation and configured router-on-a-stick inter-VLAN routing.
- **Access Control Lists (ACLs)** — Implemented Layer 3 security policies to control inter-department traffic.
- **Dynamic NAT / PAT** — Configured NAT overload to simulate public internet access for private networks.
